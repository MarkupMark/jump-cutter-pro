# Jump Cutter Pro — Privacy Policy

Last updated: 2026-07-13

Jump Cutter Pro is a browser extension that detects quiet portions of audio and changes playback speed to help users watch videos more efficiently.

## Data collection

Jump Cutter Pro does not collect, sell, or transmit browsing history, video content, audio recordings, account information, or personal information to the developer or to third-party services.

Audio analysis and playback control happen locally in the browser. The extension does not use an external analytics service, advertising service, remote database, or user account.

## Local browser storage

The extension stores its settings, keyboard shortcuts, and time-saved counters in the browser extension storage. Chrome may synchronize these settings when the user has enabled browser extension synchronization. This data is not sent to the developer.

## Permissions

- `storage`: save extension settings and counters.
- `scripting`: support the extension's media-control implementation.
- Access to HTTP and HTTPS pages: detect compatible audio/video elements and control playback on pages where the user has enabled the extension.

## Changes

If the extension's data practices change, this policy will be updated before those changes are introduced.
